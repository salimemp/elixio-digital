# Elixio Digital — Onboarding Video Script

**Target duration:** 6 minutes 30 seconds
**Languages:** English (primary) + 11 priority translations (ES, FR, DE, HI, PT, AR, UR, HE, ZH, ZH-TW, JA, KO)
**Aspect ratio:** 16:9 (1920×1080)
**Frame rate:** 30 fps
**Voiceover style:** Warm, confident, energetic. Think Apple keynote meets a friendly tutorial.

---

## Global production notes

- **Brand colors:** Gum pink (#ff90e8), purple (#7b61ff), yellow (#f1e05a), mint (#96f7d6), cyan (#23a6d5)
- **Typography:** Inter (display) + IBM Plex Serif (body, for emphasis only)
- **Logo treatment:** White wordmark on gum-purple, or gum-yellow on gum-black
- **Motion language:** Springy, playful, bouncy — 200-400ms transitions. Not corporate.
- **Music:** Soft electronic with acoustic guitar accents. 110-120 BPM. Loop-friendly.
- **Sound design:** UI sounds for clicks, transitions, whooshes. Always subtle.
- **Accessibility:** Open captions burned in. Voiceover reads every on-screen number.

### Shot list (for AI video generation)
For each scene, generate 1-3 short video segments (5-8s each, 1920×1080, 30fps) with prompts that match the visuals above. Use a consistent style: clean product screenshots, soft motion graphics, occasional live-action b-roll. Avoid stock-photo-looking AI generations.

### Voiceover script (deliverable, English)
See the inline voiceover blocks. Each scene's VO is in the `> "..."` quote.

### Subtitle / SRT file structure
Burned-in English captions throughout. For the 11 priority languages, generate SRT files with the same timestamps. Upload to YouTube as closed captions; embed on the website via `<track>` tags.
